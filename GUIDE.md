## 第1章 MOD概要

### このMODとは
**MD_TSF_Submod**は、Hearts of Iron IV のMOD **Millennium Dawn: A Beta Test Mod** に Muv-Luv シリーズの**戦術機（TSF: Tactical Surface Fighter）**を追加するサブMODです。

### 追加要素
- 60+ 機種の戦術機（国別固有機 + 汎用機）
- 軽/重/高機動/支援/特務の5カテゴリの戦術機素体
- 専用研究ツリー（基礎研究＋世代別framework＋モジュール）
- 専用師団テンプレ9種（軽/重/高機動/混成/支援/特務/特務突撃/凄ノ皇/F-47）
- 凄ノ皇（XG-70b/c/d）、F-47（試製イシュクル）などの超兵器
- 国別固有研究（武御雷・瑞鶴・月虹など）

### 動作環境
| 項目 | 推奨 |
|---|---|
| HOI4本体 | **1.18.3.0**（1.19系では現状非対応）|
| MD Beta | 2.0.0 |
| 戦術機Submod | 0.1.0 |
| OS | Windows |

### 必須・推奨MOD構成
**プレイセットに有効化するMOD：**
- ✅ Millennium Dawn: A Beta Test Mod（必須）
- ✅ MD TSF Submod（このMOD）
- ⚠️ `+JP: Millennium Dawn: A Modern Day mod`（**無効化推奨**：MD 2.0.0非対応で日本語表示崩壊の原因）
- ⭕ JLm Fonts / Japanese Map Fonts Sans（共存OK）

### MODの基本方針
このMODは **「AIが戦術機を運用しやすくする」** ことを最優先に設計されています。プレイヤー向けの自由度は最低限保ちつつ、AIが固有機体まで確実に辿り着けるシステムが組み込まれています。

## 第2章 基本コンセプト

### 戦術機とは（Muv-Luv世界観の基礎）
戦術機は対BETA戦のために開発された人型機動兵器。航空機をベースに二足歩行型の戦闘機として運用される。世代別に性能が向上し、第1世代（1990年代）から第4世代（2030年代）まで発展。

### このMODでの戦術機システム

#### 5つのカテゴリ
| カテゴリ | 役割 | 例 |
|---|---|---|
| **軽戦術機** | 速度重視・偵察突破 | F-5、撃震、不知火（旧設定）|
| **重戦術機** | 装甲・火力重視 | 撃震、陽炎、月虹、F-47 |
| **高機動戦術機** | 機動・突破力 | 不知火、F-15E、武御雷 |
| **支援戦術機** | 対空・防御特化 | 壱型丙（不知火・壱型丙）|
| **特務戦術機** | エリート・斯衛 | 瑞鶴・武御雷各型 |

#### 既存兵科との関係
| 戦術機 | 既存MD兵科との関係 |
|---|---|
| カテゴリ | 戦車（armor）と並列・独立カテゴリ |
| 工場割り当て | 全archetype × 10工場（minimum）|
| 大隊構成 | 既存兵科と組み合わせ可（大隊枠は別）|
| AI戦略 | tsf_combined_arms（混成）優先 |

### 大隊と師団の階層
```
装備（戦術機） → 大隊（10機の戦術機） → 師団（10大隊で1師団）
```
- 重戦術機師団 = 重大隊×10 = 戦術機120機 + cnc装備100個

## 第3章 研究ツリー攻略

### 全体構造

```
[基礎] basic_tsf_framework (1995年)
  ↓
[世代1] 1_light / 1_light_mod / 1_heavy framework
  ↓
[世代2] 2_light / 2_heavy / 2_maneuver framework
  ↓
[世代3] 3_light / 3_heavy / 3_maneuver framework
  ↓
[世代4] 4_framework

[特務系統] F-4JK → T00 → T00R → T00F/A/C
[超兵器系統] tsf_susanoo_concept → 各種susanoo研究
            tsf_F-47_framework → F-47武装 → F-47_systems
```

### 研究の優先度（AI factor）

本MODではAIの研究選択を最適化するため、各テックに factor を設定：

| テックの種類 | factor | 効果 |
|---|---|---|
| **前提モジュール**（armor_heavy、engine_1等） | **90** | AIが優先研究 |
| **素体framework**（tsf_X_Y_framework） | **90** | AIが優先研究 |
| **非前提モジュール**（custom系、deta_link等）| **40** | 後回し |

### 固有機体の解放条件
固有機体（撃震・陽炎・月虹など）は以下の**両方**を満たすと配布される：
1. 該当世代の **素体framework** を研究済み
2. 対応する **前提モジュール**（5-8個）を研究済み

→ 順番は問わない（モジュールが先でも素体framework が先でも可）

### 効率的な研究順（プレイヤー視点）

#### 序盤（1995-2005年）
1. `basic_tsf_framework`（即解放）
2. 1世代モジュール一括（factor=90で自動的に優先される）
3. `1_heavy_framework` or `1_light_framework`（撃震/F-4Jなど）

#### 中盤（2005-2015年）
4. 2世代モジュール
5. `2_*_framework` → 陽炎・吹雪等の解放
6. 特務系統開始（F-4JK → T00）→ 瑞鶴・武御雷

#### 終盤（2015年以降）
7. 3世代モジュール
8. `3_*_framework` → 月虹・不知火・弐型・武御雷R
9. 4世代（2035年）→ 04式/10式
10. 超兵器：tsf_susanoo_concept (2000)、tsf_F-47_framework (2040)

### 隠しテック（hidden tech）
固有機体・汎用機・支援機は `allow = no` の隠しテックで配布される：
- `tsf_variant_jap_1h` → 撃震解放
- `tsf_variant_jap_2h` → 陽炎解放
- `tsf_common_f15` → 汎用F-15解放
- `tsf_support_3l` → 3世代軽支援機解放

プレイヤーは研究画面に出ない。on_actionsで自動配布される。

## 第9章 AI挙動の理解

### AIの研究優先順位
1. **前提モジュール**（factor=90）を最優先
2. **素体framework**（factor=90）と並行
3. **非前提モジュール**（factor=40）は後回し

→ 結果：撃震・陽炎などの**固有機体に確実に辿り着く**

### AIの師団訓練比率
| 師団 | role_ratio |
|---|---|
| 混成戦術機師団 | 40% |
| 軽/重/高機動 | 各15% |
| 支援 | 8% |
| 特務 | 5% |
| 特務突撃 | 3% |
| 凄ノ皇 | 微量 |

### AIの装備選択ロジック
1. 同archetypeで priority 最高を優先
2. priority同値なら stat 比較（対空・装甲・突破等）
3. 国別配布変種が優先（固有機体）

#### priority体系
- 固有機体：100（武御雷C/瑞鶴Cも100）
- 武御雷A/瑞鶴A：75
- 武御雷F/瑞鶴F：50
- 武御雷R/瑞鶴R：25（最希少）
- 汎用機：10（圧倒的劣位）

### 軽戦術機偏重問題
**症状**：AIが軽戦術機ばかり作る  
**原因**：混成師団40%×軽大隊4 + 軽師団15%×10大隊 = 軽の需要が多い  
**対策**：重戦術機師団テンプレが付与されていれば改善（AI需要が均衡）

### 自動修復機能
- 9種の師団テンプレが**毎月チェック**され、廃止・改名されたら**AIにのみ自動再付与**
- プレイヤーは自由に編成改正可能

## 第10章 プレイヤー専用テクニック

### 素体からの自由設計
全12種の世代別子素体が `is_buildable=yes` で出ます：

## 第11章 トラブルシューティング

### 症状別対処

#### 「重戦術機師団テンプレが無い」
**原因**：古いセーブでフラグが立った状態  
**対処**：自動補完（on_monthly）で1ヶ月以内に復活、または：
```
clr_country_flag tsf_heavy_template_fix
```

#### 「素体が出ない/解放されない」
**原因**：以前の is_buildable=no が原因  
**対処**：HOI4再起動で is_buildable=yes が反映される

#### 「固有機体（撃震・陽炎等）が出ない」
**原因**：前提モジュール研究未完了  
**対処**：basic_framework + 該当world研究 + 前提モジュール5-8個を完了

#### 「AIが軽戦術機ばかり作る」
**原因**：重戦術機師団テンプレが付与されていない、または比率不均衡  
**対処**：on_monthlyで自動補完待ち、または手動でテンプレ復活

#### 「装備リストに『戦術機』のままで重/軽が表示されない」
**原因**：archetype が MVLV_TSF_chassis のままの装備  
**対処**：F-47の場合は本MODで chassis_heavy に変更済み

#### 「HOI4起動で文字化け・MD本体loc崩壊」
**原因**：古い`+JP MOD`（HOI4 1.17対応）がプレイセットに残ってる  
**対処**：その`+JP MOD`を無効化

#### 「ゾンビモード重い」
**原因**：Multiple Locations + Extreme でゾンビ大量発生  
**対処**：Single Location + Normal/Hardに変更

## 第12章 Event Horizon シナリオ攻略

### Event Horizon とは
MD Beta 2.0.0 で追加された**特殊シナリオ**。事件性のあるイベントが連鎖的に発生する。

### 戦術機MODとの相性
- ✅ 基本的に共存OK
- ⚠️ シナリオ独自イベントは戦術機MOD非対応
- ⚠️ ゲームルールでEvent Horizon有効化していない場合は無視可

### 戦術機編成の最適解
Event Horizon中の戦闘は通常戦争と異なる特徴：
- 短期決戦が多い → **高機動戦術機師団**有効
- 都市戦が多い → **重戦術機**で要塞突破
- 制空権重要 → **支援戦術機**で対空

### 段階別攻略
| フェーズ | 推奨師団 | 目的 |
|---|---|---|
| 序盤（〜2005）| 軽戦術機師団（撃震/汎用F-5）| 機動防衛 |
| 中盤（〜2015）| 混成師団 + 重師団 | 戦線確立 |
| 終盤（2015〜）| 特務+凄ノ皇 | 決定的勝利 |

### 失敗パターン
- 戦術機準備が間に合わない → 開幕から研究優先
- マンパワー不足 → 戦術機師団は2000人/師団で消耗激しい
- 装備不足 → 工場70+確保必須

## 第13章 ゾンビモード攻略

### ゾンビモードとは
MD Beta 2.0.0 に内蔵された機能。ゲームルールで有効化するとZOM国家（ゾンビ）がスポーンし、全人類連合と戦う。

### ゲームルール設定の選び方

#### Zombie Game Mode
- **No**：通常ゲーム
- **Yes**：ゾンビモード有効

#### Spawn Location（最重要）
| 設定 | 初期師団数 | 重さ |
|---|---|---|
| **AFRICA** | 126師団 | **軽量** |
| EUROPE | 約180師団 | 中 |
| MULTIPLE（5地点） | 210師団 | 中 |
| **EAST_ASIA**（3地点）| **378師団** | **重量** |

#### Difficulty
| 設定 | 効果 | 推奨環境 |
|---|---|---|
| NORMAL | 標準 | バランス |
| **HARD** | やや強化 | **おすすめ** |
| EXTREME | 大幅強化・週次増殖 | 高スペックPC |
| INSANE | 最強化 | 限界挑戦 |

#### Spawn Delay
| 設定 | 発生タイミング |
|---|---|
| **None（デフォルト）** | ゲーム開始即時 |
| 1 Year | 1年後 |
| 2 Years | 2年後 |
| **3 Years** | **3年後（準備時間取れる）**|

#### Anti-Zombie Coalition
- Enabled（全主要国参加）：標準
- Disabled：個別対応

### 戦術機 vs ゾンビ戦術

| 状況 | 推奨師団 | 理由 |
|---|---|---|
| **大量集団対応** | **軽戦術機師団** | 手数で殲滅、組織率回復早い |
| **要塞化ゾンビ** | **重戦術機師団** | 突破・装甲突破 |
| **包囲・機動** | **高機動戦術機** | 速度で囲い込み |
| **後方守備** | **戦術機支援師団** | 対空・防御重視 |
| **エリート防衛** | **特務戦術機** | 高ステータス |
| **エンドゲーム** | **凄ノ皇/F-47** | 圧倒的火力 |

### 国別初動

#### 日本
1. ゲーム開始 → 即 basic_framework 研究
2. 撃震・陽炎を量産
3. 本土防衛先行 → 大陸介入
4. 武御雷で精鋭部隊

#### 米国
1. 反ゾンビ連合主導
2. F-5/F-15で物量
3. 本土死守、海軍展開
4. F-47解放後（2040）が真価

#### 露中
1. 早期攻勢で領土確保
2. ロシア：MiG-21/Su-27で物量
3. 中国：J-10/J-11で密集陣

### パフォーマンス対策
| 軽量化策 | 効果 |
|---|---|
| Spawn Location → AFRICA | 師団数 1/3 |
| Difficulty → NORMAL | 増殖無し |
| グラフィック設定 → 低 | 描画負荷減 |
| ゲーム速度5 | 体感速度向上 |
| バックグラウンドアプリ終了 | CPU余裕 |

### ゾンビMOD（Doomsday）との関係
- **不要**：MD Beta 2.0.0 にゾンビモードが**完全内蔵**
- スタンドアロン版（1.17対応）は競合の原因
- **必ず無効化**
